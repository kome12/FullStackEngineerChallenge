# Technology Overview

## Backend
- Node.js / Express
- Database is MongoDB

## Frontend
- Angular
- Material

## Quickstart
I usually separate out the frontend and backend into 2 projects so 2 separate commands / terminal windows will be needed.

### Backend
`cd backend`

`npm install`

`npm run start`

Opens on `http://localhost:3000`

### Frontend
`cd frontend`

`npm install`

`ng serve`

Opens on `http://localhost:4200`
